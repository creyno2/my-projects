# 🌟 Charity's Projects

A portfolio site hosted on **GitHub Pages** showcasing my projects.

## 🌐 Live Site
Once deployed: `https://creyno2.github.io/my-projects/`

## 🛠️ Tech Stack
- Plain HTML + Tailwind CSS (via CDN)
- Hosted free on GitHub Pages

## 🚀 How to Run Locally
Just open `index.html` in your browser — no build step needed!

## 📦 Deploying Updates
```bash
git add .
git commit -m "Update projects page"
git push origin main
```
GitHub Pages auto-deploys within ~60 seconds. 🎉
